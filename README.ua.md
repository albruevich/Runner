Проста endless runner гра, створена у **Lens Studio** для Snapchat.

Біжіть вперед, уникайте перешкод, збирайте нагороди та намагайтеся побити свій рекорд.

## Вимоги

Щоб відкрити, редагувати або запустити проєкт локально, потрібно:

- **Lens Studio** (рекомендована остання версія)
- Snapchat акаунт (необов’язково, для тестування на мобільному)
- Windows або macOS

## Геймплей

- Переміщення між 3 доріжками
- Стрибки через перешкоди
- Збір нагород за очки
- 3 серця (життя)
- Швидкість гри зростає з часом
- Рекорд зберігається локально
- Миттєвий рестарт після Game Over

## Керування

### Desktop / Editor

- **Left Arrow** — рух вліво  
- **Right Arrow** — рух вправо  
- **Up Arrow** або **Space** — стрибок  
- **Mouse Drag Left / Right** — рух  
- **Mouse Drag Up** — стрибок  
- **Будь-який Tap / Click / Input** на стартовому екрані — почати гру

### Mobile

- **Swipe Left** — рух вліво  
- **Swipe Right** — рух вправо  
- **Swipe Up** — стрибок  
- **Tap / Swipe** на стартовому екрані — почати гру
